CodeQuest 2024

A project which measures your Jira tasks' skill level and rewards your effort

Featuring:

  • AI based Task hardness measuring
  • Developer XP bars
  • Rewarding & Lvl ups
  • Fancy design

How it works?

Task hardness is measured

Developer checks skill level

Developer accepts the task

Developer is rewarded with XP

Rewards can be redeemed upon lvl up
